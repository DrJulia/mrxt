
# MRXT #

The Multi-robot eXploration Tool MRXT is a mobile robot simulator for testing Exploration and SLAM algorithms.

# Installation #

MRXT has been tested in Ubuntu 12.04.

Requirements:
* Qt4
* Loki
* OpenCV

To install MRXT:
qmake
make
make install

To run MRXT:
mrxt
